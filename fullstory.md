# Full Story Campaign: Valdris Chronicle (Extended Branching Edition)

Dokumen ini adalah versi pengembangan penuh untuk 3 chapter campaign Valdris.
Fokus desain:
- percabangan banyak,
- banyak aksi/pertarungan,
- ada cabang loot item (weapon, potion, relic, armor),
- semua cabang tetap kembali ke jalur cerita utama (main spine).

---

## Sistem Naratif Inti

### Main Spine (Tidak Boleh Hilang)
1. **Chapter 1**: Pelarian The Pale Hand + jejak awal.
2. **Chapter 2**: Jaringan bayangan + Shadow Knight.
3. **Chapter 3**: Serangan ibukota + final confrontation.

### Rule Percabangan
Semua cabang dibagi jadi:
- **Major Branch**: mengubah rute, nuansa konflik, dan resource party.
- **Minor Branch**: side action (loot, rescue, puzzle, duel).
- **Merge Node**: titik balik ke main spine.

---

## Item & Resource Loop (Cabang Loot)

Item ditemukan di cabang, lalu ikut sampai chapter selanjutnya.

### Item Kunci
- **Ironfang Blade** (weapon): bonus agresi melee.
- **Aegis Buckler** (armor): mengurangi dampak gagal di 1 check fisik.
- **Vial of Ember** (consumable): sekali pakai, reroll check STR/DEX.
- **Moonwater Flask** (consumable): pulih HP naratif (menghapus penalti luka).
- **Sigil Lens** (utility): bonus informasi untuk cabang investigasi.
- **Blackvault Key Fragment** (quest): membuka shortcut penting chapter 2.
- **Crimson Core Shard** (relic): opsi unik di chapter 3.

### Rule Kembali ke Cerita Utama
Mau menang/kalah di cabang loot, node akan tetap mengarah ke merge node utama.
Perbedaannya: status party, item, dan flavor ending.

# CHAPTER 1 — Escape of the Pale Hand (Extended)

## Main Spine Node
- Alarm penjara pecah.
- Party pilih respons awal.
- Tracking ke Hutan Kelam.
- Konfrontasi camp The Pale Hand.
- End chapter: The Pale Hand lolos / terdesak / kehilangan relik.

## Branch Set A — Respon Kota

### A1. Gerbang Kota (Combat Line)
- Encounter: mercenary line + Orc vanguard.
- Checks:
  - STR untuk break formation,
  - CON untuk menahan serangan balik.
- Side loot branch:
  - **A1-Loot-01**: peti suplai prajurit -> dapat **Aegis Buckler**.
- Merge ke Forest Edge.

### A2. Gorong-gorong (Hazard Line)
- Encounter: gas trap, sludge tunnel, rat swarms.
- Checks:
  - DEX menghindar jebakan,
  - WIS membaca jalur aman.
- Side loot branch:
  - **A2-Loot-01**: ruangan teknisi lama -> **Vial of Ember**.
- Merge ke Forest Edge.

### A3. Atap Kota (Recon Line)
- Encounter: rooftop chase, sniper cultist.
- Checks:
  - DEX balance,
  - INT memetakan jalur pelarian.
- Side loot branch:
  - **A3-Loot-01**: sarang pemanah -> **Ironfang Blade**.
- Merge ke Forest Edge.

## Branch Set B — Forest Operations

### B1. Tracking Fisik
- INT check membaca jejak.
- Fail -> tersesat -> wolf ambush.

### B2. Tracking Spirit/Aura
- WIS check membaca gelombang sihir.
- Success -> shortcut camp, bonus inisiatif.

### B3. Side Branch: Hermit Hut
- Opsional saat hutan.
- Pilihan:
  - tolong pertapa melawan beast,
  - curigai pertapa agen kultus.
- Reward:
  - baik: **Moonwater Flask**,
  - buruk: info palsu (waktu terbuang).
- Merge ke Camp Approach.

## Branch Set C — Camp Confrontation

### C1. Frontal Assault
- STR/CON high-risk battle.
- Potensi luka tinggi, kill pressure tinggi.

### C2. Stealth Relic Grab
- DEX/INT precision run.
- Jika punya Sigil Lens (later), stealth jauh lebih aman.

### C3. Diplomatic Probe
- CHA/WIS: negosiasi + baca kebohongan.
- Bisa buka lore tentang "Ritual Xal’thor" lebih cepat.

### C4. Side Duel Branch (Opsional)
- Tantang champion camp 1v1.
- Menang -> moral musuh drop, dapat **Blackvault Key Fragment**.
- Kalah -> tetap lanjut, tapi party kena penalti stamina.
- Merge ke Chapter 1 End.

## Ending Chapter 1 (Multi Outcome)
- **E1-Tactical Win**: relik sempat direbut sebagian.
- **E2-Pyrrhic Win**: menang tapi luka berat.
- **E3-Information Win**: negosiasi gagal tangkap, tapi dapat intel ritual.
- **E4-Hard Loss**: The Pale Hand lolos dominan.

Semua ending -> masuk Chapter 2, beda start modifiers.

# CHAPTER 2 — Shadows of Xal’thor (Extended)

## Main Spine Node
- Kota penuh paranoia.
- Jejak menuju Sanctum Xal’thor.
- Party menembus area ritual.
- Boss: Shadow Knight.
- End chapter: peta serangan ke ibukota.

## Branch Set A — Investigasi Politik vs Underground

### A1. Loyalist Route
- Dewan perang, arsip militer.
- Checks: CHA/INT.
- Side loot:
  - armory cache -> **Reinforced Vambrace** (armor utility).
- Merge node: Northern Expedition.

### A2. Underworld Route
- pasar gelap, kode penyelundup.
- Checks: INT/DEX.
- Side loot:
  - contraband vault -> **Sigil Lens**.
- Merge node: Northern Expedition.

### A3. Cult Route
- katakombe, ritual chamber.
- Checks: WIS/CON (mental pressure).
- Side loot:
  - altar fragment -> **Crimson Core Shard**.
- Merge node: Northern Expedition.

## Branch Set B — Northern Expedition

### B1. Main Road Siege
- mass combat checkpoint.
- STR route.

### B2. Ice Tunnel Hazard
- traversal + environmental collapse.
- DEX/CON route.

### B3. Side Branch: Frozen Reliquary
- Puzzle ruangan es.
- Jika berhasil -> **Moonwater Flask x2**.
- Jika gagal -> minor injury, tetap merge.

### B4. Side Branch: Fallen Knight Grave
- keputusan moral: ambil senjata makam atau hormati.
- Ambil: dapat **Runeblade Hilt** (weapon part).
- Hormati: dapat blessing (1 auto-stabilize fail).
- Merge ke Sanctum Gate.

## Branch Set C — Sanctum Interior

### C1. Front Hall Breach
- fight + trap.
- reward: jalan cepat ke altar.

### C2. Service Shaft Sneak
- stealth + puzzle wheel.
- reward: flank advantage.

### C3. Archive Chamber (Side Knowledge)
- baca catatan ritual asli.
- reveal: Shadow Knight cuma medium, bukan sumber utama.
- Merge ke Boss Prelude.

## Boss Complex — Shadow Knight

Tiga phase boss:
1. **Phase 1**: blade duel (STR/DEX).
2. **Phase 2**: dark pulse (CON/WIS).
3. **Phase 3**: core decision (attack core vs focus boss).

### Loot/Consequence Boss
- Jika hancurkan core: dapat **Core Coordinates** (bonus chapter 3).
- Jika hanya kalahkan boss: dapat **War Map Fragment**.

## Ending Chapter 2 (Multi Outcome)
- **E1-Decisive Win**: Shadow Knight runtuh + data lengkap.
- **E2-Trade Win**: menang tapi core sebagian lepas.
- **E3-Survival Exit**: terdesak tapi bawa info cukup.
- **E4-Catastrophic Escape**: Sanctum runtuh, info minim.

Semua ending -> Chapter 3, dengan opsi strategi berbeda.

# CHAPTER 3 — True Hand of Valdris (Extended)

## Main Spine Node
- Ibukota diserang.
- Menembus garis pertahanan.
- Naik menara.
- Konfrontasi final The Pale Hand.
- Ending campaign multi-branch.

## Branch Set A — Entry to Capital Core

### A1. Frontline Breakthrough
- STR + CON attrition.

### A2. Wall Climb Infiltration
- DEX + CON precision.

### A3. Side Branch: Rescue Civilians
- menyelamatkan warga dari kebakaran.
- reward: moral bonus + informasi jalur pendek.
- fail: waktu habis, tapi merge tetap.

### A4. Side Branch: Royal Treasury Annex
- opsional loot sebelum menara.
- reward:
  - **Royal Elixir** (heal status),
  - atau **Sunsteel Spearhead** (weapon part).
- merge ke Tower Ascent.

## Branch Set B — Tower Ascent

### B1. Spiral Stair Gauntlet
- trap + elite guard.

### B2. Broken Lift Shaft
- climbing + timing.

### B3. Mirror Hall Puzzle (Side)
- puzzle identitas The Pale Hand.
- sukses -> buka dialog route lebih kuat.
- merge ke Final Platform.

## Final Confrontation — The Pale Hand

Pilihan utama:
1. **Annihilate** (battle heavy).
2. **Disarm Artifact** (technical strike).
3. **Redeem** (social-psychological).

### Expanded Sub-Branches

#### Route 1: Annihilate
- Phase combat 3 gelombang.
- Jika punya Ironfang/Runeblade parts -> kill window lebih besar.
- Ending:
  - clean kill,
  - mutual destruction,
  - kill with collateral.

#### Route 2: Disarm Artifact
- DEX/INT/WIS chain checks.
- Jika punya Crimson Core Shard -> stabilisasi energi lebih mudah.
- Ending:
  - artifact sealed,
  - artifact shattered,
  - unstable detonation.

#### Route 3: Redeem
- CHA/WIS route + bukti masa lalu.
- Jika sebelumnya ambil branch archive/mirror hall, dialog options lebih kuat.
- Ending:
  - surrender,
  - sacrifice redemption,
  - fake surrender twist.

## Campaign Ending Matrix

### Heroic Endings
- **Golden Dawn**: kerajaan selamat, warga pulih, party jadi legenda.
- **Silent Crown**: menang tanpa banyak darah, reformasi dimulai.

### Bittersweet Endings
- **Ashen Victory**: menang tapi ibukota rusak parah.
- **Broken Banner**: musuh kalah, tapi salah satu pahlawan gugur naratif.

### Dark Endings
- **Crimson Night**: artifact meledak, Valdris lumpuh.
- **False Peace**: Pale Hand kabur diam-diam; ancaman lanjut.

## Merge Design (Agar Tetap Utuh)

Setiap branch punya `return_to_main` secara desain:
- branch loot -> merge node chapter concern.
- side combat -> merge node sebelum boss.
- optional puzzle -> unlock bonus, bukan blok progress.

Dengan begitu:
- cerita tetap utuh,
- replayability tinggi,
- developer mudah tambah chapter 4+.

---

## Blueprint Chapter Lanjutan (Scalable)

Untuk menambah chapter berikutnya:
1. Buat `node_001_intro` sebagai start wajib.
2. Tambahkan minimal 3 major branches.
3. Tambahkan minimal 3 side loot branches.
4. Pastikan semua branch punya merge node.
5. Tutup dengan `effect` end node + hook chapter berikut.

Contoh hook Chapter 4:
- fragmen artifact aktif kembali di benua timur,
- nama baru: **Order of the Hollow Star**.

---

## Catatan Implementasi untuk JSON Scenario

Agar kompatibel engine saat ini:
- gunakan ID konsisten (`node_...`),
- `dice_check` wajib punya `text`, `difficulty`, `required_ability`,
- hindari field liar di luar schema aktif,
- gunakan `effect` untuk node ending chapter,
- loot/state bisa disimpan via metadata room atau inventory store.

---

Valdris Chronicle versi ini didesain untuk terasa seperti campaign hidup: banyak cabang, banyak aksi, banyak reward, tapi tetap satu benang merah cerita yang kuat.
